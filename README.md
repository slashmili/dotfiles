
Clone repo
==========
```
$ git clone git@github.com:slashmili/dotfiles.git .dotfiles
$ cd .dotfiles && git submodule update --init --recursive
```


Setup vim setting
=================
```
$ ln -s ~/.dotfiles ~/.vim
$ echo "source ~/.dotfiles/.vimrc" >> ~/.vimrc
$ echo "source ~/.dotfiles/.bashrc >> ~/.bashrc
```
