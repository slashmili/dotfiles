Clone repo
==========
```
$ git clone https://github.com/slashmili/dotfiles.git ~/.dotfiles
```


Setup vim settings
=================
```
$ ln -s ~/.dotfiles/.vim ~/.vim
$ echo "source ~/.dotfiles/.vimrc" >> ~/.vimrc
```

Bashrc settings
===============
```
$ echo "source ~/.dotfiles/.bashrc" >> ~/.bashrc
```
