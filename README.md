Clone repo
==========
```
$ git clone https://github.com/slashmili/dotfiles.git ~/.dotfiles
```


Setup ~vim~ Neovim settings
=================
```
$ ln -s ~/.dotfiles/.config/nvim ~/.config/
```

Bashrc settings
===============
```
$ echo "source ~/.dotfiles/.bashrc" >> ~/.bashrc
```


Packages
=========
```
$ brew install rg
```
